# To-Do-App-Python
A simple console-based To-Do application written in Python.  
This project focuses on practicing basic programming concepts such as lists, functions, loops, and user interaction.

## Features

- Add tasks with a title, description, and status
- View tasks marked as incomplete
- Delete tasks using a keyword
- Menu-driven command-line interface

## Usage Instructions

1. Run the Python script.
2. Choose one of the following options:
   - `1` → Add a task
   - `2` → View incomplete tasks
   - `3` → Delete a task using a keyword
   - `4` → Exit the program
